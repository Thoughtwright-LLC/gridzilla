    <h1 id=“label-Gridzilla”><a href=“Gridzilla.html”>Gridzilla</a><span><a href=“#label-Gridzilla”>&para;</a> <a href=“#documentation”>&uarr;</a></span></h1>

    <p><a href=“Gridzilla.html”>Gridzilla</a> is a griding dsl for Rails projects which allows users to list and perform actions on rows of data.</p>

    <h1 id=“label-Installation+Steps”>Installation Steps<span><a href=“#label-Installation+Steps”>&para;</a> <a href=“#documentation”>&uarr;</a></span></h1>

    <p>Run the generator to place the images, javascript, stylesheet, and initializer in the correct locations with this command:</p>

    <p>rails g gridzilla:install</p>

    <p>Next, add this line to your config/routes.rb file:</p>

    <p>match ‘/gridzilla/:controller/:action’ =&gt; ‘#index’, :gridzilla =&gt; true</p>

    <p>Last, add this line above the initialize! call to your config/environment.rb file:</p>

    <p>require ‘will_paginate’</p>
